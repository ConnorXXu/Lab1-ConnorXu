network protocols

  • FTP (File Transfer Protocol) is a network protocol for transmitting files between computers over Transmission Control Protocol/Internet Protocol (TCP/IP) connections.
  • HTTP is a protocol for fetching resources such as HTML documents.
  • The Internet Protocol (IP) is the method for sending data from one device to another across the internet. Every device has an IP address that uniquely identifies it and enables it to communicate with and exchange data with other devices connected to the internet.
  • TCP is a transport protocol that is used on top of IP to ensure reliable transmission of packets.
